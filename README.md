# Cairo Programming Assignments

This repository contains 62 unique assignments for students learning Cairo programming and smart contract development. Each student has a dedicated folder with a practical assignment covering beginner to intermediate concepts.

# Structure

- Each folder: `<Student_Name>_<Topic_Name>`
- After the assignment each folder should have:
  - `guide.md` (detailed description of the file)
  - Starter `.cairo` file

# Assignment Rules

- Each assignment is unique and must be completed individually.

- You have to complete the assignment using 'cairo' programming language.

- Get help from Cairo book

- Plagiarism will result in disciplinary action.
- Submit only your own work.
- Follow the coding standards and naming conventions.



# Contribution Instructions

- Each student must work only in their assigned folder.
- Do not modify other students' folders.
- Follow the assignment instructions in your `README.md`.
- Ensure your code is syntactically correct and runs with Scarb.
- Commit only your own work.
- The Pull Request should target:master branch of the original class repository

-After opening the PR:DO NOT merge it yourself
-The instructor/admin will review and merge it into the main repository

-Your submission is considered COMPLETE only when:

-A Pull Request is opened to the main repository
-Your code appears in the PR
-The PR is visible in the original repository

-If your work exists only on your own GitHub account and no PR is submitted to the main repository, your assignment will NOT be counted.

# Submission Guidelines

- Complete your assignment in your folder.
- After submitting remove your name from the folder and add only the topic name.
- Star the repository
- Ensure your code runs without errors.
- Push your changes to the repository before the deadline.
- Include comments as well in the code.
- Include detailed description of the topic you have studied add all about it in guide.md

# Grading 

- Each and every step given above have some weightage in the assignment

- This is going to be the highest weightage assignment among all of your assignments.
- You can use AI but you are required to explain everything you have done during evaluations.
- Non serious students will definately get negative marking
- Exceptionally brilliant work will be appreciated will absolute extra marking
